## 🎯 Como Usar Esta Estrutura

### Para Encontrar Responsáveis
1. **Por tipo de tarefa:** Veja a tabela acima
2. **Por documento:** Cada arquivo tem um mantenedor
3. **Por issue:** Issues devem ser atribuídas ao responsável da área

### Para Atualizar Documentos
1. Identifique o responsável pela área
2. Notifique via Discord/Telegram/Email
3. Ou envie um PR e marque o responsável para review

### Para Adicionar Novos Responsáveis
1. Crie uma pasta com seu nome: `docs/responsaveis/<nome>/`
2. Copie o template: `RESPONSAVEL.md`
3. Atualize com suas informações e responsabilidades
4. Adicione à tabela acima
